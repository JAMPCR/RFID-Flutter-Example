# Zebra RFID Flutter Plugin Example for Android/iOS
 The source code in this repository is available to use for the purpose of intergrating Zebra's RFID devices with Flutter.  

 The demo application provided to test the Flutter plugin is for testing purposes only.

## Documentation 
 Documention for writing Flutter plugins are available online from several resources

## Prerequisites
 * Flutter installed
 * iOS Builds ( Mac, xCode )
 * Android Builds ( Mac/Windows, Android Studio )
